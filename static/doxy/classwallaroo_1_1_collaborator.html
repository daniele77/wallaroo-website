Detailed Description

template<typename T, typename P = mandatory, template< typename E, typename Allocator=std::allocator< E > > class Container = std::vector>
class wallaroo::Collaborator< T, P, Container >

This represents a "collaborator" of a "part" that you can link with another "part".

If the part1 has the collaborator1 linked to part2, part1 will basically get a pointer to part2.

Template Parameters
TThe type of the Part contained
PThis represents the kind of Collaborator (mandatory if you must link a part, optional if you can leave this collaborator unlinked, collection if you can link many parts to this collaborator)
ContainerIf P = collection, this represents the std container the Collaborator will derive from.
