Detailed Description

template<typename T, typename P = mandatory, template< typename E, typename Allocator=std::allocator< E > > class Container = std::vector>
class wallaroo::Plug< T, P, Container >

This represents a "plug" of a "device" that you can "plug" into another "device".

If the device1 has the plug1 plugged to device2, device1 will basically get a pointer to device2.

Template Parameters
TThe type of the Device contained
PThis represents the kind of Plug (mandatory if you must wire a device, optional if you can leave this plug unwired, collection if you can wire many devices to this plug)
ContainerIf P = collection, this represents the std container the Plug will derive from.
